Grocery Pruchasing Patterns

...the three dimensional representation in Orbit mode

Each glyph in the 3D scatter plot represents a purchase. Purchases are binned according to size of the unit (i.e. pint, pound, dozen), food group, and food. Each glyph is colored and sized according to the total amount spent.

Click and drag the mouse inside the image to control the viewpoints.

...about the visualization

This plot is a view of spending in the grocery list according to one of the 3 categorical variables: food group, item, and unit.

If you use the mouse to pick on the plot, the items contributing to the picked category will appear in the other plot.

...about the visual program

This visual program shows the application of categorical tools to the visualization of a grocery list.
